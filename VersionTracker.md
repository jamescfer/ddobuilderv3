# Version Tracker

This file records every released version of **DDOBuilder**. It is updated
**automatically** by the [Auto Release on Version Bump](.github/workflows/release.yml)
workflow every time a version bump is merged to `main` and a release is
published — there is no need to edit it by hand.

Each row captures the version that was released, the date it was published
(UTC), the git tag, and the commit the release was built from.
